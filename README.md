# Personal Homepage Draft

This folder now contains a ready-to-open static homepage scaffold:

- `index.html`: the main page content
- `stylesheet.css`: the page style

Quick edits you may want to make before publishing:

1. Replace the placeholder contact links in `index.html`.
2. If you want a real portrait, replace the `YH` monogram block with an image.
3. Fill in your biography, publications, projects, and affiliations section by section.
4. The original template's extra project folders are still in this repo; they are harmless now because the new homepage does not link to them, but you can clean them up later if you want a smaller repository.

To preview locally, open `index.html` in a browser or serve this folder with any simple static file server.
